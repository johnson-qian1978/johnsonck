# OpenClaw Web UI 设计文档

## 📋 项目概述

**目标**：参考腾讯 WorkBuddy 的桌面端界面设计，为 OpenClaw 打造一个现代化、易用且功能强大的 Web 控制界面。

**设计原则**：
- 简洁直观，降低学习成本
- 信息密度高但不拥挤
- 响应式设计，支持多设备
- 保持 OpenClaw 的核心特色

---

## 🎨 界面布局

### 整体结构：三栏布局

```
┌─────────────────────────────────────────────────────────────────┐
│  Top Bar (可选：全局搜索、设置、用户头像)                        │
├──────────┬────────────────────────────┬─────────────────────────┤
│          │                            │                         │
│  左侧    │       中间主内容区          │      右侧面板           │
│  导航栏   │    (对话/工作区)           │   (任务/状态/上下文)     │
│  240px   │        自适应宽度           │      280px             │
│          │                            │                         │
├──────────────────────────────────────┴─────────────────────────┤
│  底部输入区 (固定高度)                                           │
└─────────────────────────────────────────────────────────────────┘
```

## 📍 左侧导航栏 (240px)

### 顶部区域
- **Logo + 品牌名**：OpenClaw (可点击返回首页)
- **搜索框**：搜索任务、会话、文件

### 主要功能入口 (图标 + 文字)
1. **🆕 新建任务** - 创建新会话/任务
2. **🔧 Claw** - 核心功能入口
3. **👥 专家** - 子代理管理
4. **🎯 技能** - 已安装技能列表
5. **🧩 插件** - 插件管理
6. **⚙️ 自动化** - Cron 任务/定时任务

### 任务/会话列表
- 按时间排序的最近会话
- 显示标题 + 时间戳 + 状态图标
- 支持快速切换
- 可折叠/展开

### 底部用户信息
- 用户头像 + 昵称
- 当前状态（在线/忙碌/离线）

### 对话区域

#### 🎯 核心特性：思考过程可视化（参考 WorkBuddy）

每条 AI 回复采用**结构化展示**，清晰呈现完整的思考和执行过程：

**1. 步骤分解卡片**
```
┌─────────────────────────────────────────────┐
│ 🔍 网页搜索                                │
│ WorkBuddy OpenClaw 配置 coding.dashscope... │
│ └─ 已完成 (0.8s)                           │
└─────────────────────────────────────────────┘
       ↓
┌─────────────────────────────────────────────┐
│  网页获取                                │
│ OpenClaw-大模型服务平台百炼...              │
│ └─ 已完成 (1.2s)                           │
└─────────────────────────────────────────────┘
       ↓
┌─────────────────────────────────────────────┐
│ 👁️ 已读取                                  │
│ C:\Users\27151\.openclaw\openclaw.json     │
│ └─ 进行中...                               │
└─────────────────────────────────────────────┘
```

**2. 状态标识**
- ✅ **已完成** - 绿色对勾 + 耗时
- 🔄 **进行中** - 旋转动画 + 进度条
- ⏸️ **等待确认** - 黄色警告 + 操作按钮
- ❌ **失败** - 红色叉号 + 错误信息 + 重试按钮

**3. 可折叠的任务列表**
- 默认展开最近 3-5 个步骤
- 点击"任务列表 5/5 ▼"查看全部
- 支持快速跳转到特定步骤
- 每个步骤可独立展开/收起详情

**4. 文件/资源操作可视化**
- 👁️ **已读取** - 显示文件路径 + 行数范围
- ✏️ **已修改** - 显示变更内容摘要
- 📝 **已创建** - 显示保存路径
- 🗑️ **已删除** - 显示删除项（带确认）

**5. 自然语言总结**
- 在技术细节上方用**粗体**突出关键发现
- 使用口语化表达（"找到根本原因了！"）
- 重要结论单独成段，便于快速扫描

### 视图 1️⃣: 任务清单（默认）
- **待办任务列表**（带复选框）
- **优先级标识**（高/中/低，彩色圆点）
- **截止时间提醒**（倒计时显示）
- **可拖拽排序**
- **任务分组**（按项目/日期/状态）
- **完成进度**（X/Y 个任务已完成）

### 视图 2️⃣: 产物面板 📦
**顶部 Tab 切换**：`产物` | `全部文件` | `变更` | `预览`

#### 产物标签页
- **生成的文件列表**（图标 + 文件名 + 大小）
- **分类展示**：
  - 📄 文档（.md, .txt, .pdf）
  - 💻 代码（.js, .py, .ts, .html）
  - 📊 数据（.json, .csv, .xlsx）
  - 🖼️ 媒体（.png, .jpg, .mp4）
- **快速操作**：
  - 👁️ 预览
  - 📋 复制路径
  - 📤 下载
  - 🗑️ 删除
- **按时间排序**（最新在前）

#### 全部文件标签页
- 工作区完整文件树
- 支持折叠/展开文件夹
- 文件修改时间戳
- Git 状态标识（新增/修改/删除）

#### 变更标签页
- **本次会话的文件变更历史**
- Diff 对比（行级高亮）
- 回滚到指定版本
- 导出变更日志

#### 预览标签页（核心特性！）
**智能预览窗口** - 根据文件类型自动适配：

- **📊 数据报表**：
  - 表格可视化（ sortable columns）
  - 图表渲染（折线图、柱状图、饼图）
  - KPI 卡片（大数字 + 趋势箭头）
  - 示例：投资组合日报、交易记录表

- **💻 代码文件**：
  - 语法高亮
  - 行号显示
  - 代码折叠
  - 一键复制

- **📄 文档内容**：
  - Markdown 渲染
  - 目录导航
  - 内链跳转

- **🖼️ 图片/视频**：
  - 缩略图 → 大图查看
  - 视频播放器（支持播放/暂停/进度条）
  - 缩放/旋转

- **🌐 网页/HTML**：
  - 内嵌 iframe 预览
  - 响应式尺寸调整
  - 外部链接在新标签打开

**预览窗口特性**：
- 🔙 **面包屑导航**：显示文件路径
- ⬅️️ **左右切换**：快速浏览多个产物
- 📥 **下载按钮**：一键导出
- 🔗 **分享链接**：生成可访问 URL（可选）
- 🖥️ **全屏查看**：最大化预览区域

## 🔧 交互细节

### 加载状态
- 骨架屏（Skeleton）用于内容加载
- 旋转 Spinner 用于操作等待
- 进度条用于文件上传/下载

### 动画效果
- 页面切换：淡入淡出 `0.2s ease`
- 消息出现：从下往上滑入 `0.3s ease-out`
- 按钮点击：轻微缩放 `0.1s`
- 悬停效果：背景色变化 + 轻微上浮

### 响应式设计
- **桌面端 (>1200px)**: 完整三栏布局
- **平板端 (768-1200px)**: 隐藏右侧面板，可手动展开
- **移动端 (<768px)**: 单栏布局，导航收起到汉堡菜单

## 🛠️ 技术选型建议

### 前端框架
- **推荐**: React 18 + TypeScript
- **备选**: Vue 3 + Vite

### UI 组件库
- **Ant Design** - 企业级组件丰富
- **Shadcn/ui** - 现代化设计，高度可定制
- **Tailwind CSS** - 原子化 CSS，灵活快速

### 状态管理
- **Zustand** - 轻量简单
- **Redux Toolkit** - 复杂场景

### 实时通信
- **WebSocket** - 与 OpenClaw Gateway 通信
- **Socket.IO** - 备选方案（支持降级）

### 代码高亮
- **Prism.js** 或 **Highlight.js**

### Markdown 渲染
- **react-markdown** + **remark-gfm**

## 📝 注意事项

1. **保持轻量化**：避免过度设计，首屏加载时间 < 2s
2. **渐进增强**：核心功能优先，高级功能逐步添加
3. **可访问性**：支持键盘导航、屏幕阅读器
4. **安全性**：XSS 防护、CSP 策略、敏感操作二次确认
5. **与现有 UI 兼容**：保留原有 Control UI 作为备选
