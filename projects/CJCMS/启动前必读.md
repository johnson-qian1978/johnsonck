### 2. Maven 3.8+ ❌ (未安装)

**检查是否已安装**:
```bash
mvn -version
```

**如果未安装**，有以下选择：

#### 方案 A: 安装 Maven（推荐）
1. 下载：https://maven.apache.org/download.cgi
2. 解压到：`C:\Program Files\Apache\maven`
3. 配置环境变量：
   - `MAVEN_HOME = C:\Program Files\Apache\maven`
   - PATH 添加 `%MAVEN_HOME%\bin`
4. 验证：`mvn -version`

#### 方案 B: 使用 IDE（最简单）⭐
**IntelliJ IDEA** (推荐):
1. 下载：https://www.jetbrains.com/idea/download/
2. 安装后打开 IDEA
3. File → Open → 选择 `E:\CJCMS`
4. 等待依赖自动下载
5. 右键 `CJCmsApplication.java` → Run

**Eclipse**:
1. 下载：https://www.eclipse.org/downloads/
2. File → Import → Existing Maven Projects
3. 选择 `E:\CJCMS`
4. 右键项目 → Run As → Spring Boot App
