# 修复说明 - GUI 版本错误

## ❌ 问题描述

运行 GUI 版本时报错：
```
获取信息失败：<x_wmi: WMI returned a syntax error: you're probably running 
inside a thread without first calling pythoncom.CoInitialize[Ex]>
```

## 🔧 问题原因

WMI（Windows Management Instrumentation）在多线程环境中使用时，需要先初始化 COM 组件。之前的版本没有在后台线程中初始化 COM，导致错误。

## ✅ 解决方案

已修复！在 `SystemInfoWorker` 线程类中添加了：

```python
def run(self):
    try:
        # 在线程中初始化 COM
        pythoncom.CoInitialize()
        
        # ... 获取系统信息 ...
        
    finally:
        # 清理 COM
        try:
            pythoncom.CoUninitialize()
        except:
            pass
```
