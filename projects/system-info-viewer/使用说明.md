# Windows 系统信息查看工具 - 使用说明

## 📦 打包完成！

已成功创建可执行文件（EXE），无需安装 Python 即可运行！

---

## 🚀 使用方法（推荐）

### 方法 1：直接双击 EXE（最简单）

- **命令行版本**：双击 `SystemInfoViewer.exe`
- **图形界面版本**：双击 `SystemInfoViewer_GUI.exe`

### 方法 2：使用启动脚本（带错误日志）

- **命令行版本**：双击 `启动 - 命令行版.bat`
- **图形界面版本**：双击 `启动 -GUI 版.bat`

> ✅ 启动脚本会自动记录错误到 `error.log` 文件

## 🔍 如果运行失败

### 1. 查看错误日志

双击 `查看错误日志.bat` 或者打开 `error.log` 文件查看具体错误。

### 2. 常见错误及解决方法

#### 错误：找不到 SystemInfoViewer.exe
```
解决：确保 EXE 文件在同一目录下
```

#### 错误：应用程序无法正常启动 (0xc000007b)
```
解决：需要安装 Visual C++ Redistributable
下载地址：https://aka.ms/vs/17/release/vc_redist.x64.exe
```

#### 错误：被杀毒软件拦截
```
解决：这是误报，请将 EXE 添加到杀毒软件白名单
或者暂时关闭杀毒软件后运行
```

#### 错误：需要管理员权限
```
解决：右键点击 EXE → 以管理员身份运行
```

### 3. 联系支持

如果问题仍未解决，请：
1. 打开 `error.log` 文件
2. 截图错误信息
3. 联系技术支持

## ⚠️ 注意事项

1. **首次运行可能较慢**：EXE 首次运行需要解压，请耐心等待
2. **杀毒软件误报**：PyInstaller 打包的程序可能被误报，请添加白名单
3. **文件大小**：EXE 文件较大（12MB/41MB），因为包含了 Python 运行环境
4. **兼容性**：Windows 10/11 系统
