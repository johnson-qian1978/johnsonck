# Windows 系统信息查看工具 - 更新说明

## 📦 版本更新

### v2.0.0 (2026-03-12 13:20) - 台账专用版

#### ✅ 新增功能

1. **USB 设备历史记录**
   - 从注册表获取所有曾经连接过的 USB 存储设备
   - 显示设备名称、序列号
   - 显示最后连接时间（如果可获取）
   - 即使设备已拔出也能看到历史记录

2. **实体打印机过滤**
   - 自动过滤虚拟打印机（如 PDF 打印机、WPS PDF 等）
   - 只显示真实的物理打印机
   - 避免台账填写错误

3. **网络接口分类**
   - **有线网卡**：明确标注以太网接口
   - **无线网卡**：明确标注 WLAN/WiFi 接口
   - **虚拟适配器**：单独分类，不混淆视听
   - **连接状态**：显示 [已连接] 或 [未连接]

#### 📋 台账信息优化

根据非涉密信息设备台账填写需求，优化了以下信息：

| 台账项目 | 显示位置 | 说明 |
|----------|----------|------|
| 计算机型号 | 【计算机信息】 | 用于填写设备型号 |
| 序列号 | 【计算机信息】 | BIOS 序列号/主板序列号 |
| 使用人 | - | 需手动填写 |
| 网卡 MAC 地址 | 【网络信息】 | 有线/无线分别显示 |
| IP 地址 | 【网络信息】 | 只显示实际连接的 |
| USB 端口使用历史 | 【USB 设备历史】 | 所有连接过的 USB 设备 |
| 打印机连接 | 【实体打印机】 | 只显示真实打印机 |

## ⚠️ 注意事项

### USB 历史记录限制

1. **最后连接时间**：部分 Windows 版本可能无法获取精确时间
2. **已格式化设备**：如果 USB 设备被格式化，序列号可能变化
3. **注册表清理**：如果清理过注册表，部分记录可能丢失

### 打印机识别

以下类型的打印机会被自动过滤（不显示）：
- Microsoft Print to PDF
- WPS PDF
- OneNote
- Fax/传真
- Adobe PDF
- 其他虚拟 PDF 打印机

### 网络接口

- **只显示已连接的接口**：未连接的网卡不会显示 IP
- **虚拟适配器单独分类**：如 aTrust、VMware 等
- **回环地址不显示**：127.0.0.1 不显示
