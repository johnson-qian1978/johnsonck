# Windows 系统信息查看工具 🖥️

一个专业的 Windows 系统硬件信息获取工具，可快速查看计算机的硬件配置、系统信息、网络信息等。

---

## 📦 文件说明

| 文件名 | 说明 | 使用方式 |
|--------|------|----------|
| `system_info_viewer.py` | 命令行版本主程序 | `python system_info_viewer.py` |
| `system_info_viewer_gui.py` | GUI 图形界面版本 | `python system_info_viewer_gui.py` |
| `run.bat` | 命令行版启动脚本 | 双击运行 |
| `run_gui.bat` | GUI 版启动脚本 | 双击运行 |
| `test_result.txt` | 测试结果输出 | 查看示例输出 |
| `TEST_REPORT.md` | 完整测试报告 | 查看详细测试 |

---

## 🚀 快速开始

### 方法 1：双击运行（推荐）

- **命令行版本**：双击 `run.bat`
- **图形界面版本**：双击 `run_gui.bat`

### 方法 2：命令行运行

```bash
# 命令行版本
python system_info_viewer.py

# GUI 版本
python system_info_viewer_gui.py
```

## ✨ 功能特性

### 🔍 信息获取

- ✅ **计算机信息** - 制造商、型号、序列号
- ✅ **BIOS 信息** - 版本、序列号
- ✅ **主板信息** - 产品名、序列号
- ✅ **CPU 信息** - 型号、核心数、线程数、频率
- ✅ **内存信息** - 总容量
- ✅ **硬盘信息** - 型号、序列号、容量
- ✅ **网络信息** - MAC 地址、IP 地址
- ✅ **USB 历史** - USB 存储设备记录
- ✅ **打印机** - 打印机列表和状态
- ✅ **系统时间** - Windows 安装时间、启动时间

### 💾 数据导出

- 📄 **文本格式** - 人类可读的详细报告
- 📊 **JSON 格式** - 机器可读的结构化数据

## 📋 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 |
| Python | 3.8+ |
| 必需库 | wmi, pywin32, psutil |
| GUI 版本 | PyQt5 |

---

## 🔧 依赖安装

如果缺少依赖，运行以下命令安装：

```bash
pip install wmi pywin32 psutil PyQt5
```

---

## 📸 GUI 界面功能

图形界面版本提供以下功能：

1. **📊 系统概览** - 查看系统基本信息汇总
2. **🔧 硬件信息** - 树形结构展示硬件详情
3. **🌐 网络信息** - 显示所有网络接口信息
4. **📱 外部设备** - USB 设备和打印机信息

### 界面按钮

- **🔄 刷新信息** - 重新获取最新系统信息
- **💾 保存结果** - 保存为文本文件
- **📄 导出 JSON** - 导出为 JSON 格式
- **❌ 退出** - 关闭程序

## ⚠️ 注意事项

1. **权限要求**：某些硬件信息可能需要管理员权限
2. **WMI 限制**：仅在 Windows 系统上可用
3. **编码**：程序使用 UTF-8 编码
4. **临时文件**：所有输出保存在当前目录
