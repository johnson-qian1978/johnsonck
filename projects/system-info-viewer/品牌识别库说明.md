# USB 设备和打印机品牌识别库

## 📦 品牌库说明

本工具内置了 USB 设备和打印机的品牌识别库，可以自动识别设备制造商。

---

## 🔍 USB 设备品牌识别

### 支持的品牌类型

#### 存储设备品牌（60+ 个）
- **主流品牌**：三星、闪迪、金士顿、威刚、创见、雷克沙等
- **硬盘品牌**：西部数据、希捷、东芝、日立、英特尔等
- **内存品牌**：美光、英睿达、芝奇、海盗船、十铨等
- **其他品牌**：索尼、飞利浦、惠普、戴尔、联想等

#### 外设品牌（40+ 个）
- **鼠标键盘**：罗技、雷蛇、赛睿、海盗船等
- **音频设备**：创新、雅马哈、森海塞尔、铁三角、BOSE 等
- **专业设备**：PreSonus、M-Audio、NI、Roland 等

#### 手机品牌（20+ 个）
- **国产品牌**：华为、小米、OPPO、VIVO、荣耀、一加等
- **国际品牌**：苹果、三星、谷歌、摩托罗拉、诺基亚等

### 识别原理

1. **设备名称匹配**：从设备名称中提取品牌关键词
2. **序列号匹配**：部分品牌有特定序列号格式
3. **智能匹配**：不区分大小写，支持多种命名格式

### 示例

| 设备名称 | 识别结果 |
|----------|----------|
| `SAMSUNG USB Device` | 三星 (Samsung) |
| `SanDisk Ultra` | 闪迪 (SanDisk) |
| `Kingston DataTraveler` | 金士顿 (Kingston) |
| `APPLE iPhone` | 苹果 (Apple) |
| `HUAWEI MATE` | 华为 (Huawei) |

## 🔧 扩展品牌库

如果需要添加新的品牌，可以编辑 `device_brands.py` 文件：

```python
# 添加 USB 设备品牌
USB_BRANDS = {
    'YOUR_BRAND_KEYWORD': '品牌名称 (中文)',
    # ...
}

# 添加打印机品牌
PRINTER_BRANDS = {
    'YOUR_BRAND_KEYWORD': '品牌名称 (中文)',
    # ...
}
```

## ⚠️ 注意事项

1. **识别准确率**：基于关键词匹配，可能存在误识别
2. **未知品牌**：无法识别的设备显示为"未知品牌"
3. **OEM 设备**：部分 OEM 设备可能识别为原始品牌
4. **更新品牌库**：定期更新以支持新品牌
