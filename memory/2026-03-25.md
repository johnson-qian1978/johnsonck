## 💡 关键学习

1. **Oracle 编码问题**
   - ❌ 不要直接用 PowerShell 命令行执行包含中文的 SQL 脚本
   - ✅ 在 PL/SQL Developer 中手动执行，或使用 N'' 前缀的 Unicode 字符串
   - ✅ 设置 `ALTER SESSION SET NLS_LANGUAGE = 'SIMPLIFIED CHINESE'`

2. **Maven 环境配置**
   - 路径中有空格时必须用引号包裹或用环境变量引用
   - 正确方式：`& "$env:MAVEN_HOME\bin\mvn.cmd"`

3. **Spring Security 配置**
   - 静态资源必须显式添加到 permitAll 列表
   - `.requestMatchers("/dashboard.html", "/admin-list.html").permitAll()`

4. **启动流程规范化**
   - 先检查并释放端口
   - 再启动新进程
   - 使用独立窗口显示日志（用户可见）
