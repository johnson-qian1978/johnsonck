## GH_BMD Management System

Created a web-based CRUD application for the GH_BMD Oracle table.

### Tech Stack
- **Frontend**: Vue 3 + Element Plus (single HTML file)
- **Backend**: Node.js + Express
- **Database**: Oracle 11g (using Thick mode via oracledb)

### Key Features
- Left sidebar navigation (Wuhan Medical Mutual Aid style)
- Search/filter by name and type (4 types: unit name, ID number, over-retirement age, different-unit retiree)
- Data table with pagination
- Add/Edit/Delete operations
- Auto-generated UUID for new records (32-char hex format matching existing data)

### Issues Fixed

2. IPv6 connection issues - forced 127.0.0.1
3. Field name case sensitivity (Oracle returns uppercase)
4. Pagination display logic
5. Auto-generate UUID for new records instead of manual input
