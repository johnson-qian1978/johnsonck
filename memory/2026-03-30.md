### 遇到的问题及解决

1. **`forbiddenfruit` 编译卡住**
   - 原因: Windows 不支持，需要在 Linux/WSL 环境运行
   - 解决: 使用 Docker 容器构建

2. **`extensions_config.json` 是目录**
   - 原因: Docker 挂载时自动创建
   - 解决: 删除目录，创建空 JSON 文件 `{}`

3. **JSON BOM 编码问题**
   - 原因: Windows PowerShell 写入 UTF-8 BOM
   - 解决: 使用 `[System.IO.File]::WriteAllText(path, "{}", [System.UTF8Encoding]::new($false))`
