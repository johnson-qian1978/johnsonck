### 正确的集成方式

根据 DeerFlow 官方文档：
1. **DeerFlow → OpenClaw (MCP)**：DeerFlow 配置 `extensions_config.json`，调用 OpenClaw MCP Server
2. **DeerFlow 独立 IM**：DeerFlow 直接配置 Telegram/Slack/Feishu，不需要 OpenClaw
3. **OpenClaw MCP Server**：`openclaw mcp serve` 暴露通道供其他 MCP 客户端调用
